# Black-Scholes and Options
## Black-Scholes Fundamentals
During the summer of 2025, I watched Robert J Shiller's Yale course, _Intro to Finance_, and I was interested in the Black-Scholes equation. I created these initial pieces of code to better understand the equations behind pricing calls, puts, and the Greeks.

## Newton-Raphson and Bisection - Implied Volatility
I implemented the Newton-Raphson method to find implied volatility, and I put in a fallback Bisection method in case it did not converge within the set number of iterations. This helps me plot my volatility surface, and I learned a lot about numerical methods.
